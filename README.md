# Golden Road LoL (英雄联盟黄金之路)

**[English](#english) | [中文](#chinese)**

<a name="english"></a>
## English

Golden Road LoL is a web-based strategy draft game for League of Legends esports fans. Each run challenges you to build a championship roster by spinning the wheels to draft players from historical teams (2013-2025). Can your drafted team achieve the legendary "Golden Road"?

### 🌟 Features
- **Wheel Draft System:** Spin to randomly select a Region -> Team -> Year, then pick an available player for your 5-man roster.
- **Deep Historical Data:** Features 26 iconic teams, 93 rosters, and 465 player records from the LPL, LCK, LEC, and LCS, each with hidden historical performance ratings (1-99).
- **Two Game Modes:**
  - *Golden Road*: Simulates the ultimate gauntlet (Spring Split -> MSI -> Summer Split -> Worlds).
  - *MSI Challenge*: A shorter, high-stakes run through the Mid-Season Invitational format.
- **Simulation Engine:** Uses a logistic regression-based probability engine to determine Match outcomes, making every BO5 intense and unpredictable.
- **Social Sharing:** Generate high-quality graphical share cards of your run results, copy roster text, or post directly to X (Twitter).
- **Fully Bilingual:** Seamless real-time switching between English and Simplified Chinese.

### 🚀 Tech Stack
- **Frontend:** Pure HTML5, Vanilla JavaScript, and CSS3.
- **Design:** Modern dark mode, glassmorphism UI, interactive animations, and responsive layout.
- **Zero Dependencies:** No backend or framework overhead. Runs directly in any modern browser.

### 🎮 How to Play
1. **Clone or Download** the repository.
2. Open `index.html` in your web browser.
3. Select your mode, spin the wheels, and start building your legendary roster!

## 中文 (Chinese)

**黄金之路 LoL** 是一款专为英雄联盟电竞粉丝打造的网页端策略选秀挑战游戏。玩家需要通过转盘随机抽取历史知名战队（2013-2025年），并从中挑选选手组建你的梦之队。你的阵容能否完成传说中的“黄金之路”？

### 🌟 核心特色
- **转盘选秀机制**：依次随机抽取 赛区 -> 战队 -> 年份，并从该阵容中选择一名选手填补你的首发五人名单。
- **海量电竞史料**：收录了 LPL、LCK、LEC 和 LCS 四大赛区共 26 支传奇战队、93 个历史阵容以及 465 名选手的真实评分数据（评分 1-99）。
- **双重挑战模式**：
  - *黄金之路模式*：模拟完整的年度大满贯征程（春季赛 -> MSI -> 夏季赛 -> 全球总决赛）。
  - *MSI 挑战模式*：快节奏的季中冠军赛（入围赛 -> 小组赛 -> 淘汰赛 -> 决赛）。
- **赛事模拟引擎**：基于逻辑回归概率算法模拟 BO5/BO3 赛果，高评分阵容胜率更高，但电竞魅力就在于充满悬念。
- **一键分享战绩**：游戏结束后自动生成精美的高清战绩卡片，支持保存图片、复制纯文本和一键发送到 X (Twitter)。
- **原生双语支持**：内置原生中英双语，一键无缝切换。

### 🚀 技术架构
- **前端实现**：纯 HTML5、Vanilla原生 JavaScript 和 CSS3。
- **设计美学**：采用现代极简暗色主题、毛玻璃质感 (Glassmorphism)、顺滑的交互动画与全响应式布局。
- **零依赖架构**：无任何后端服务器或笨重框架，纯前端运行，双击 `index.html` 即可畅玩。

### 🎮 如何游玩
1. **克隆或下载** 本项目代码。
2. 在浏览器中直接双击打开 `index.html`。
3. 选择你想挑战的模式，转动转盘，开始属于你的电竞传奇！
